API key setup

You need an Anthropic API key to generate quiz questions and summaries.

  1. Go to console.anthropic.com and sign in
  2. Navigate to API Keys and create a new key
  3. Paste it into the Anthropic API Key field at the top of this page
🔒 Your key is stored only in your browser session (sessionStorage) and is sent directly to Anthropic's API. It is never stored on any server or shared with anyone. It will be cleared when you close your browser tab.
Two ways to build a course

Multiple files: Drop several .md files — each becomes one named module. Reorder them freely before generating.

Single file: Use ## headings — each section becomes a module automatically.

Tips for better questions
  • Include specific numbers, limits, and thresholds in your docs
  • Document procedures as numbered steps
  • Include FAQ-style content — AI turns these into great questions
  • Tables with specs generate strong technical questions
  • Aim for 300–800 words per module section
